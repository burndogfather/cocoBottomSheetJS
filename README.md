# cocoBottomSheetJS
Similar to the bottom sheet of iOS, it can be implemented with JavaScript🔥
   
[NOTICE] This component is written in VanillaJS!
[Reference] This component supports reactive UI!

### Try demo!
  
- ```<input/>``` 태그만으로 이용
- 검색기능 지원
- 무한스크롤방식의 페이징 지원
- 호출-응답 커스텀 가능
- 사용자 선택에 따른 콜백 작성가능
- 사전선택 값 설정가능  
- 키보드입력 지연시간 설정가능
- 키보드입력 정규식조건 설정가능
